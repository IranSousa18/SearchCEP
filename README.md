# BConsultar CEP

Sistema web desenvolvido para consulta de endereços a partir de um CEP utilizando a API ViaCEP.

## Funcionalidades

- Consulta de CEP em tempo real
- Validação de entrada de dados
- Tratamento de CEPs inválidos ou inexistentes
- Busca através do botão ou tecla Enter
- Interface responsiva
- Exibição dinâmica dos dados do endereço

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES6+)
- API ViaCEP

## Objetivo

Projeto desenvolvido com foco em praticar consumo de APIs REST, programação assíncrona com Async/Await, manipulação do DOM e boas práticas de desenvolvimento Front-End.
